# EC 410K / EC 610I: Machine Learning in Economics

**Wilfrid Laurier University — Spring 2026**

Instructor: **M. Jahangir Alam, PhD**  
Course codes: **EC 410K / EC 610I**

## About the Course

This course introduces machine learning methods and their applications in empirical economics, with emphasis on:

- Prediction and causal inference
- Supervised and unsupervised learning
- Regularization and high-dimensional models
- Tree-based methods and model interpretability
- Macroeconomic and financial forecasting
- Natural language processing and generative AI
- Empirical replication and reproducible research

Students will use Python to replicate empirical economics research and develop machine-learning extensions of published studies.
